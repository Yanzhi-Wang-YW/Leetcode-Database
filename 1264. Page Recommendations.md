```md
1264. Page Recommendations  
Table: Friendship

+---------------+---------+
| Column Name   | Type    |
+---------------+---------+
| user1_id      | int     |
| user2_id      | int     |
+---------------+---------+
(user1_id, user2_id) is the primary key (combination of columns with unique values) for this table.
Each row of this table indicates that there is a friendship relation between user1_id and user2_id.
 

Table: Likes

+-------------+---------+
| Column Name | Type    |
+-------------+---------+
| user_id     | int     |
| page_id     | int     |
+-------------+---------+
(user_id, page_id) is the primary key (combination of columns with unique values) for this table.
Each row of this table indicates that user_id likes page_id.
 

Write a solution to recommend pages to the user with user_id = 1 using the pages that your friends liked. It should not recommend pages you already liked.

Return result table in any order without duplicates.

The result format is in the following example.

 

Example 1:

Input: 
Friendship table:
+----------+----------+
| user1_id | user2_id |
+----------+----------+
| 1        | 2        |
| 1        | 3        |
| 1        | 4        |
| 2        | 3        |
| 2        | 4        |
| 2        | 5        |
| 6        | 1        |
+----------+----------+
Likes table:
+---------+---------+
| user_id | page_id |
+---------+---------+
| 1       | 88      |
| 2       | 23      |
| 3       | 24      |
| 4       | 56      |
| 5       | 11      |
| 6       | 33      |
| 2       | 77      |
| 3       | 77      |
| 6       | 88      |
+---------+---------+
Output: 
+------------------+
| recommended_page |
+------------------+
| 23               |
| 24               |
| 56               |
| 33               |
| 77               |
+------------------+
Explanation: 
User one is friend with users 2, 3, 4 and 6.
Suggested pages are 23 from user 2, 24 from user 3, 56 from user 3 and 33 from user 6.
Page 77 is suggested from both user 2 and user 3.
Page 88 is not suggested because user 1 already likes it
```

```py
import pandas as pd

def page_recommendations(friendship: pd.DataFrame, likes: pd.DataFrame) -> pd.DataFrame: 
    user_one_friend = friendship.loc[friendship['user1_id'] == 1] 
    user_one_friend = user_one_friend.rename(columns={'user1_id': 'user_name', 'user2_id': 'friend_name'}) 
    user_one_more_friend = friendship.loc[friendship['user2_id'] == 1] 
    user_one_more_friend = user_one_more_friend[['user2_id', 'user1_id']] 
    user_one_more_friend = user_one_more_friend.rename(columns={'user2_id': 'user_name', 'user1_id': 'friend_name'}) 
    all_friend = pd.concat([user_one_friend, user_one_more_friend]) 
    all_friend = all_friend.drop_duplicates(subset=['user_name', 'friend_name']) 
    all_friend = all_friend.merge(likes, how='inner', left_on='friend_name', right_on='user_id') 
    all_friend = all_friend[['page_id']] 
    all_friend = all_friend.rename(columns={'page_id': 'recommended_page'}) 
    all_friend = all_friend.drop_duplicates(subset=['recommended_page']) 
    user_one_like = likes.loc[likes['user_id'] == 1] 
    user_one_like = user_one_like.drop_duplicates(subset=['page_id']) 
    output = all_friend.merge(user_one_like, how='left', left_on='recommended_page', right_on='page_id', indicator=True) 
    output = output.loc[output['_merge'] == 'left_only' ] 
    output = output[['recommended_page']]              
    return output
``` 
