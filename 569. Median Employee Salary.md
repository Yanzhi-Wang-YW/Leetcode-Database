```py 
import pandas as pd 
import numpy as np 

def median_employee_salary(employee: pd.DataFrame) -> pd.DataFrame:  
    employee = employee.sort_values(['company', 'salary'])
    employee['rn'] = employee.groupby('company').cumcount() + 1  
    employee['total'] = employee.groupby('company')['id'].transform('count')  
    employee['check_median_position'] = np.where(
        (employee['total'] / 2 <= employee['rn']) & (employee['rn'] <= employee['total'] / 2 + 1), 
        1, 
        0 
    ) 
    employee = employee.loc[employee['check_median_position'] == 1, ['id', 'company', 'salary']] 
    return employee 
```

### Step 5: Add the Median Logic (`Added custom 1`)

This is where the heavy lifting happens. The code adds an `IsMedian` column using a logical check based on whether the total count is odd or even:

* **If Odd (e.g., Company C has 5 rows):** It divides the total by 2, rounds down, and adds 1. (5 / 2 = 2.5 -> 2 + 1 = **Row 3**).
* **If Even (e.g., Company A has 6 rows):** A true median of an even set averages the two middle numbers. It grabs *both* middle rows: (6 / 2 = **Row 3**) AND (6 / 2 + 1 = **Row 4**).

If a row matches these middle numbers, it gets a `1`. Otherwise, it gets a `0`

  else if Number.Mod([TotalCount], 2) = 0 and ([Running Count] = [TotalCount] / 2 or [Running Count] = [TotalCount] / 2 + 1) then 1 
  else 0
