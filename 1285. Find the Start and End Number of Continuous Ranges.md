```md
1285. Find the Start and End Number of Continuous Ranges  
Table: Logs

+---------------+---------+
| Column Name   | Type    |
+---------------+---------+
| log_id        | int     |
+---------------+---------+
log_id is the column of unique values for this table.
Each row of this table contains the ID in a log Table.
 

Write a solution to find the start and end number of continuous ranges in the table Logs.

Return the result table ordered by start_id.

The result format is in the following example.

 

Example 1:

Input: 
Logs table:
+------------+
| log_id     |
+------------+
| 1          |
| 2          |
| 3          |
| 7          |
| 8          |
| 10         |
+------------+
Output: 
+------------+--------------+
| start_id   | end_id       |
+------------+--------------+
| 1          | 3            |
| 7          | 8            |
| 10         | 10           |
+------------+--------------+
Explanation: 
The result table should contain all ranges in table Logs.
From 1 to 3 is contained in the table.
From 4 to 6 is missing in the table
From 7 to 8 is contained in the table.
Number 9 is missing from the table.
Number 10 is contained in the table
``` 

```py 
import pandas as pd 
import numpy as np 

def find_continuous_ranges(logs: pd.DataFrame) -> pd.DataFrame: 
    logs = logs.sort_values(['log_id'], ascending=[True])  
    logs['prev_log_id'] = logs['log_id'].shift(1) 
    logs['diff'] = logs['log_id'] - logs['prev_log_id']
    logs['diff'] = logs['diff'].fillna(value=0) 
    logs['is_new_group'] = np.where(
        logs['diff'] != 1, 
        1, 
        0
    ) 
    logs['group_num'] = logs['is_new_group'].cumsum() 
    output = logs.groupby('group_num').agg(
        start_id=('log_id', 'min'), 
        end_id=('log_id', 'max') 
    ).reset_index() 
    output = output[['start_id', 'end_id']]          
    return output  
```
