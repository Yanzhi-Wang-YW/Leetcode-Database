```md
1205. Monthly Transactions II  
Table: Transactions

+----------------+---------+
| Column Name    | Type    |
+----------------+---------+
| id             | int     |
| country        | varchar |
| state          | enum    |
| amount         | int     |
| trans_date     | date    |
+----------------+---------+
id is the column of unique values of this table.
The table has information about incoming transactions.
The state column is an ENUM (category) of type ["approved", "declined"].
Table: Chargebacks

+----------------+---------+
| Column Name    | Type    |
+----------------+---------+
| trans_id       | int     |
| trans_date     | date    |
+----------------+---------+
Chargebacks contains basic information regarding incoming chargebacks from some transactions placed in Transactions table.
trans_id is a foreign key (reference column) to the id column of Transactions table.
Each chargeback corresponds to a transaction made previously even if they were not approved.
 

Write a solution to find for each month and country: the number of approved transactions and their total amount, the number of chargebacks, and their total amount.

Note: In your solution, given the month and country, ignore rows with all zeros.

Return the result table in any order.

The result format is in the following example.

 

Example 1:

Input: 
Transactions table:
+-----+---------+----------+--------+------------+
| id  | country | state    | amount | trans_date |

| 101 | US      | approved | 1000   | 2019-05-18 |
| 102 | US      | declined | 2000   | 2019-05-19 |
| 103 | US      | approved | 3000   | 2019-06-10 |
| 104 | US      | declined | 4000   | 2019-06-13 |
| 105 | US      | approved | 5000   | 2019-06-15 |
+-----+---------+----------+--------+------------+
Chargebacks table:
+----------+------------+
| trans_id | trans_date |
+----------+------------+

Output: 
+---------+---------+----------------+-----------------+------------------+-------------------+
| month   | country | approved_count | approved_amount | chargeback_count | chargeback_amount |
``` 

```py
import pandas as pd 
import numpy as np 

def monthly_transactions(transactions: pd.DataFrame, chargebacks: pd.DataFrame) -> pd.DataFrame: 
    transactions['trans_date'] = pd.to_datetime(transactions['trans_date']).dt.strftime('%Y-%m') 
    chargebacks['trans_date'] = pd.to_datetime(chargebacks['trans_date']).dt.strftime('%Y-%m') 
    chargebacks = chargebacks.rename(columns={'trans_date': 'chargebacks_trans_date'}) 
    chargebacks = chargebacks.rename(columns={'trans_id': 'id'})  
    transactions = transactions.merge(chargebacks, how='left', on='id') 
    transactions['is_approved'] = np.where(
        transactions['state'] == 'approved', 
        1, 
        0 
    )
    transactions['is_approved_amount'] = np.where(
        transactions['state'] == 'approved', 
        transactions['amount'], 
        0 
    ) 
    agg_transactions = transactions.groupby(['trans_date', 'country']).agg(
        approved_count=('is_approved', 'sum'), 
        approved_amount=('is_approved_amount', 'sum')
    ).reset_index() 
    agg_transactions = agg_transactions.rename(columns={'trans_date': 'month'}) 
    agg_chargebacks = transactions.groupby(['chargebacks_trans_date', 'country']).agg(
        chargeback_amount=('amount', 'sum'), 
        chargeback_count=('id', 'count') 
    ).reset_index() 
    agg_chargebacks = agg_chargebacks.rename(columns={'chargebacks_trans_date': 'month'}) 
    output = agg_chargebacks.merge(agg_transactions, how='outer', on=['month', 'country'])  
    output = output.fillna(0) 
    output = output.loc[(output['chargeback_amount'] != 0) | (output['chargeback_count'] != 0) | (output['approved_count'] != 0) | (output['approved_amount'] != 0)]    
    output = output[['month', 'country', 'approved_count', 'approved_amount', 'chargeback_count', 'chargeback_amount']]              
    return output
``` 
